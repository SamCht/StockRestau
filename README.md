# Project Title
StockRestau
Application C# pour la gestion de stock d'un restaurant.

## Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Installing
- Sur visual studio, charger la solution StockResto.sln
- Installer DB browser for Sqlite 
- Modifier le chemin d'accès à la base Sqlite dans platDAO.cs "connectionString"


## Running the tests
- Lancer l'application,
- Insérer les champs concernant le bouton "Ajouter plat" 
- Le plat est ajouté à la base sqlite 
- Faire une recherche de plat
- Faire une suppression de plat
- Ajouter plusieurs plats et afficher tous les plats
